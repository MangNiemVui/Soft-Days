# Soft Days — Multi-page Website

Website đã được tách thành nhiều trang riêng:

- `index.html` — Trang chủ
- `travel.html` — Trang du lịch
- `vlog.html` — Trang vlog
- `study.html` — Trang chill study
- `style.css` — Toàn bộ giao diện dùng chung
- `script.js` — Menu mobile và hiệu ứng fade-in

## Cách chỉnh bài viết

Muốn thêm/bớt bài ở trang nào thì mở đúng file trang đó.

Ví dụ:
- Thêm bài du lịch: mở `travel.html`
- Thêm video vlog: mở `vlog.html`
- Thêm bài học/chill study: mở `study.html`

## Cách public

Có thể upload toàn bộ các file này lên GitHub Pages, Netlify hoặc Vercel.
File mở đầu của website là `index.html`.


## Trang mới đã thêm

- `work.html` — Trang công việc / dự án / portfolio

Muốn chỉnh nội dung công việc thì mở `work.html`.
